Section Plane Tool

  • Create section cuts effects enabling you to view geometry within a model.

Tool Operation

  • Click on a face to cut create a Section Plane entity and resulting section cut effect.

Modifier Keys

  • Arrow keys = Lock Surface Normal (Up=Blue, Left=Green, Right=Red, Down=Parallel)
  • Shift (before 1st click) = Lock tool to current orientation
