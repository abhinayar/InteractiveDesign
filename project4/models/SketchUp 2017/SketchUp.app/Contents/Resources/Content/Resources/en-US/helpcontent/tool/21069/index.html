Arc Tool

  • Draw Arc entities.

Tool Operation

  1. Click to define arc's center. Optionally, click and drag first point to set drawing plane.
  2. Move cursor to define first arc point or enter radius.
  3. Click to set first arc point.
  4. Move cursor around the protractor guide or enter angle.
  5. Click to set second arc point.
  6. Esc = Cancel operation.

  7. Tip: Click Window > Model Info > Units > Angle Units to change snapping angle.

Modifier Keys

  • Ctrl '+' or Ctrl '-' = Change the number of segments
  • Hold Shift = Lock inferences
  • Arrow keys (before 1st click) = Lock Protractor Rotation Axis direction (Up=Blue, Left=Green, Right=Red, Down=Parallel)
  • Arrow keys (after 1st click) = Lock drawing direction to specific inference direction (Up=Blue, Left=Green, Right=Red, Down=Parallel/Perpendicular)
