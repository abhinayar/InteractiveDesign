Protractor Tool

  • Measure angles and create angled Construction Line entities.

Tool Operation

  1. Place protractor's center at vertex (where two lines meet) of angle.
  2. Click to set vertex. Optionally, click and drag first point to set rotation plane.
  3. Move cursor in circle until touching start of angle (one of the lines).
  4. Click to set start of angle.
  5. Move cursor in circle until touching end of angle (other line).
  6. Click to measure angle.
  7. Esc = Cancel operation.

Modifier Keys

  • Ctrl = Toggles create construction line
  • Hold Shift = Lock inferences
  • Arrow keys (before 1st click) = Lock Protractor Rotation Axis direction (Up=Blue, Left=Green, Right=Red, Down=Parallel)
  • Arrow keys (after 1st click) = Lock drawing direction to specific inference direction (Up=Blue, Left=Green, Right=Red, Down=Parallel/Perpendicular)
