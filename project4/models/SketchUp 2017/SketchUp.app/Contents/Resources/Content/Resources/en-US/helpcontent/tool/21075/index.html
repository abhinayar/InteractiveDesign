Classifier Tool

  • Add a Type to objects

Tool Operation

  1. Click the Classifier drop-down menu*.
  2. Search or browse for the Type you want to use.
  3. Select the Type.
  4. Click the object you want to classify.

*If you don't have a classification library in your model, you will need to Import a classification library and then click the Classifier tool again.

Modifier Keys

  • Alt = Sample Type from object
  • Shift (before 1st click) = Classify a single object instance
  • Ctrl (before 1st click) = Erase Types from object
