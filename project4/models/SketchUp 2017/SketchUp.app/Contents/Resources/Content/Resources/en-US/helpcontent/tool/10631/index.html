Image Igloo Tool

  • Visually selects Matched Photo Scenes.

Tool Operation

  1. Orbit with the mouse to highlight scenes.
  2. Alternatively, use Arrow keys to highlight the scene to the left/right/up/down.
  3. Press Enter to activate the highlighted scene.

Modifier Keys

  • Shift = Arrow keys select the next scene in order.
  • Ctrl = Arrow keys select within a group of photos taken from the same point.
