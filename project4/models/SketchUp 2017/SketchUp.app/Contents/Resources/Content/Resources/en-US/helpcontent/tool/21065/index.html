2-Point Arc Tool

  • Draw 2-Point Arc entities.

Tool Operation

  1. Click to set start point of arc.
  2. Move cursor.
  3. Click at ending point of arc or enter value.
  4. Move cursor perpendicular to line to set bulge distance or enter value.
  5. Click to finish arc.
  6. Esc = Cancel operation.

Modifier Keys

  • Ctrl '+' or Ctrl '-' = Change the number of segments
  • Shift = Lock tool to current inference direction
  • Arrow keys = Lock drawing direction to specific inference direction (Up=Blue, Left=Green, Right=Red, Down=Parallel/Perpendicular)
