Circle Tool

  • Draw Circle entities.

Tool Operation

  1. Click to place center point.
  2. Move cursor away from center point to define radius.
  3. Click to finish circle.
  4. Esc = Cancel operation.

Modifier Keys

  • Ctrl '+' or Ctrl '-' = Change the number of segments
  • Hold Shift = Lock inferences
  • Arrow keys (before 1st click) = Lock Surface Normal (Up=Blue, Left=Green, Right=Red, Down=Parallel)
  • Arrow keys (after 1st click) = Lock drawing direction to specific inference direction (Up=Blue, Left=Green, Right=Red, Down=Parallel/Perpendicular)
