Rectangle Tool

  • Draw rectangular Face entities.

Tool Operation

  1. Click to set first corner.
  2. Move cursor diagonally.
  3. Click to set second corner.
  4. Esc = Cancel Operation.

Modifier Keys

  • Hold Shift = Lock inferences
  • Arrow keys = Lock Surface Normal (Up=Blue, Left=Green, Right=Red, Down=Parallel)
