Label Tool
Create label text with a leader line.

Tool Operation

Create a one-segment label:

  1. Click to set start point of leader.
  2. Move mouse to define leader length.
  3. Double-click to create text box.
  4. Type text.
  5. Click outside text box to finish label.

Create a two-segment label:

  1. Click to set start point of leader.
  2. Move mouse to define first segment length.
  3. Click to create label elbow.
  4. Move mouse to define second segment length.
  5. Click to create text box.
  6. Type text.
  7. Click outside text box to finish label.

Create curved leader:

  1. Click and hold to establish start point.
  2. Move cursor away from start point.
  3. Release mouse button.
  4. Move cursor to end point.
  5. Click and hold to set end point.
  6. Move cursor away from end point.
  7. Move cursor clockwise/counter-clockwise to set bulge.
  8. Release mouse button to finish curved line.

Modifier Keys

  • Esc = Cancel operation
  • Ctrl = Toggle label direction
  • Alt = Create second segment without horizontal constraint
