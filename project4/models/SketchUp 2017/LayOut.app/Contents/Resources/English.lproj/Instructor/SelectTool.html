Select Tool
Select entities to modify when using other tools or commands.

Tool Operation

Select Entity:

  1. Click on entity.

Move entity:

  1. Click on entity.
  2. Move cursor over entity.
  3. Click and hold mouse button.
  4. Move cursor.
  5. Release mouse button to finish move.

Rotate entity:

  1. Click on entity.
  2. Move cursor over blue rotation grip.
  3. Click and hold mouse button.
  4. Move cursor around the blue center grip.
  5. Release mouse button to finish rotate.

Scale entity:

  1. Click on entity.
  2. Move cursor over edge or corner.
  3. Click and hold mouse button.
  4. Move cursor to scale.
  5. Release mouse button to finish scale.

Copy entity:

  1. Click on entity.
  2. Press and hold the Alt key.
  3. Click and hold mouse button.
  4. Move cursor to copy.
  5. Release mouse button to finish copy.

Array entity:

  1. Copy entity (see above).
  2. Type a multiplier value (eg. typing 2x will create two total copies).
  3. Or, type a divisor value. (eg. typing in 2/ will create two total copies that are equidistant apart).

Modifier Keys

  • Select entities:
    • Shift = Add or remove entities from selection
    • Alt + Click-drag = Select without moving

  • Scale entities:
    • Shift = Scale proportionally
    • Control = Scale about center
    • Alt = Copy and array
