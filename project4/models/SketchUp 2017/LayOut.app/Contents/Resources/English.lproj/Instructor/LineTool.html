Line Tool
Draw edges or individual lines and closed shapes.

Tool Operation

Create a line:

  1. Click at start point of line.
  2. Move cursor.
  3. Click at end point of line.
  4. (optional) Move cursor.
  5. (optional) Click to create connected line.
  6. (optional) Repeat step 4 to create connected lines, or return to start point of first line to create closed shape.
  7. Double-click to end line.

Create a curved line:

  1. Click and hold to establish start point.
  2. Move cursor away from start point.
  3. Release mouse button.
  4. Move cursor to end point.
  5. Click and hold to set end point.
  6. Move cursor away from end point.
  7. Move cursor clockwise/counter-clockwise to set bulge.
  8. Release mouse button to finish curved line.
  9. Double-click to end line.

Modifier Keys

  • Esc = Cancel operation
  • Shift = Lock line to red or green axes.
