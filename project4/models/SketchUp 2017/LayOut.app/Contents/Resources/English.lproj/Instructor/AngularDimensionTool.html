Angular Dimension Tool
Draw angular dimensions.

Tool Operation

  1. Click to set the connection point of the first leg of the angle.
  2. Click to set the second point of the first leg of the angle.
  3. Click to set the connection point of the second leg of the angle.
  4. Click to set the second point of the second leg of the angle.
  5. Click to set location of the angular dimension.

Esc = Cancel operation.
