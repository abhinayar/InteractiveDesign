Select Tool
Select entities to modify when using other tools or commands. Move entities. Rotate entities. Scale entities.

Tool Operation
Select Entity:
     Click on entity.

Move entity:

  1. Click on entity.
  2. Move cursor over entity.
  3. Click and hold mouse button.
  4. Move cursor.
  5. Release mouse button to finish move.

Rotate entity:

  1. Click on entity
  2. Move cursor over blue rotation grip.
  3. Click and hold mouse button.
  4. Move cursor around the blue center grip.
  5. Release mouse button to finish move.

Scale entity:

  1. Click on entity.
  2. Move cursor over edge or corner.
  3. Click and hold mouse button.
  4. Move cursor to scale.
  5. Move mouse button to finish scale.
Release mouse button.

Modifier Keys
Select entities:
     Shift = Add or remove entities from selection set
Scale entities:
     Shift = Scale proportionally

     Command = Scale about center
